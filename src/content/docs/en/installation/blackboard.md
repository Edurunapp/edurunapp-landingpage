---
title: "Blackboard Configuration"
description: "Integration for Blackboard Learn Ultra."
---

Blackboard requires registering the application in the Anthology developer portal before linking it.

## Application Registration

1.  Enter the **Administrator Panel**.
2.  Under "Integrations", select **LTI Tool Providers**.
3.  Select **Register LTI 1.3/Advantage Tool**.
4.  Enter the **Client ID** provided by Edurun support: `edurun-prod-client-id`.

## Course Deployment

Once the tool is approved ("Approved"), teachers will be able to select it from the menu:
**Content Market** > **Edurun**.

> **Warning:** Blackboard often has strict cookie policies (SameSite). Ensure the browser allows third-party cookies for the LTI session to persist.
